What Is Web Design?

web design is all about creating the look and feel of a website.Web design, is a mix of creativity and technical skills.

This is what my setup looks like when I'm coding - lots of tabs open!

Web design involves thinking about colors, fonts, spacing, and layout. But it means considering how users will interact with the site. Will they click on that button? Can they read the text easily on their phone? These are the kinds of questions web designers ask themselves.

The Development Part - Making Things Work

If web design is about how a website looks, web development is about how it works. Development is what happens behind the scenes to make everything functional. When you click a button and something happens, that's development magic!

There are different types of web developers. Front-end developers work on what users see (using HTML, CSS, and JavaScript). Back-end developers work on the server-side, databases, and application logic. Full-stack developers do a bit of everything!
